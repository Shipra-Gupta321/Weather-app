WeatherNow is a responsive web application that allows users to search for real-time weather information of any city worldwide. It provides accurate temperature, humidity, wind speed, and weather conditions using a live weather API. The app features a clean user interface, dynamic data fetching, and error handling for invalid city searches, ensuring a smooth user experience across devices.
